# Educación — artefactos.pro

10 artefactos de IA para aprendizaje y formación.  
Color de categoría: **ámbar** `#fbbf24`

| #   | Artefacto              | Carpeta                | Descripción                                               |
| --- | ---------------------- | ---------------------- | --------------------------------------------------------- |
| E-01 | Explicador de Conceptos | `explicador/`         | 3 niveles simultáneos (niño, uni, experto) + analogía     |
| E-02 | Generador de Flashcards | `flashcards/`         | Flip 3D, sistema fácil/difícil, exportar .txt             |
| E-03 | Tutor Socrático        | `tutor-socratico/`     | Diálogo guiado por preguntas con síntesis final           |
| E-04 | Quiz Adaptativo        | `quiz-adaptativo/`     | Dificultad adaptativa 1-5, timer 30s, racha, scorecard    |
| E-05 | Jerga Técnica          | `jerga-tecnica/`       | Dos paneles original/traducción, glosario, tooltips       |
| E-06 | Simulador de Entrevista | `simulador-entrevista/`| 8 preguntas con feedback por respuesta, reporte, exportar |
| E-07 | Mentor de Idiomas      | `mentor-idiomas/`      | Chat con correcciones, vocabulario acumulado, 9 idiomas   |
| E-08 | Resumen Académico      | `resumen-academico/`   | Tema claro, 8 secciones, keyword chips, citas APA/MLA    |
| E-09 | Casos Prácticos        | `casos-practicos/`     | Caso tipo Harvard, actores, resolución opcional            |
| E-10 | Debate Socrático       | `debate-socratico/`    | Tema claro, dos tesis RED vs BLUE, falacias, examen       |

## Stack

- HTML5 + CSS3 + JavaScript ES Modules (sin frameworks)
- Groq API — modelo `llama-3.3-70b-versatile`
- Importa desde `../../../js/groq.js` y `../../../js/apikey-panel.js`
